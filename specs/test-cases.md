# DeskPet M0/M1 Test Cases

This matrix covers only M0 Base Validation and M1 Safety-Clean App Shell. It does not claim M2 asset integration or full Phase 1 completion.

Automation status values:

- `automated`: covered by a committed script and command.
- `manual`: human or command-driven smoke must be run and inspected.
- `not suitable`: review/audit case that is not a product automation candidate.
- `future automation`: good candidate, but no committed test file exists yet.

Cases marked `automated` are covered by `scripts/smoke-m1.mjs` through `pnpm smoke:m1`.
